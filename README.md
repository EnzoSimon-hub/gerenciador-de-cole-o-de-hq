### Project Overview
A robust desktop application designed for comic book collectors to manage their library efficiently. It features a dual-language interface and full data persistence.

### Technical Specifications
* **Language & UI:** Developed in **Java** using the **Swing** framework for the Graphical User Interface (GUI).
* **Architecture:** Implements the **MVC (Model-View-Controller)** pattern to separate business logic from the user interface.
* **Database:** Full integration with **MySQL** via **JDBC** for permanent data storage.
* **Key Features:**
    * **CRUD Operations:** Complete Create, Read, Update, and Delete functionalities.
    * **Advanced Search:** Tab-based search system with complex filtering (by name, code, or category).
    * **Validation:** Robust input validation to ensure data integrity.
    * **Internationalization:** Interface available in English and Portuguese.

### How to Run
